# Together — инструкция по запуску

## 1. Установка зависимостей

Открой командную строку в папке проекта и выполни:

```
pip install -r requirements.txt
```

## 2. Настройка Zerotier (делается один раз)

1. Оба скачиваете и устанавливаете Zerotier: https://www.zerotier.com/download/
2. Один из вас регистрируется на https://my.zerotier.com и создаёт сеть (кнопка "Create A Network")
3. Копируете Network ID (16 символов)
4. Оба в трее Zerotier → "Join Network" → вставляете Network ID
5. На сайте my.zerotier.com в разделе Members — авторизуете оба устройства (ставите галочку)
6. После этого у каждого будет IP вида 10.147.X.X — это и есть адреса друг друга

Вместо Zerotier можно использовать RadminVPN — принцип такой же.

## 3. Первый запуск

```
python main.py
```

Приложение появится в трее (правый нижний угол, стрелочка).
Нажми правой кнопкой → "Открыть Together".

## 4. Настройка

В окне приложения → вкладка "Настройки":
- Введи своё имя
- Введи имя партнёра  
- Введи IP партнёра (Zerotier IP, вида 10.147.X.X)
- Нажми "Сохранить"

Партнёр делает то же самое — вводит твой IP.

## 5. Сборка в .exe (опционально)

```
pyinstaller --onefile --windowed --name Together --icon=assets/icon.ico main.py
```

Готовый Together.exe будет в папке dist/

```

├── requirements.txt     # Зависимости

│   ├── tray.py          # Иконка в трее

│   └── autostart.py     # Автозапуск с Windows
└── ui/
    └── window.py        # Главное окно
```
